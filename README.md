# Dbank

## Run Locally

Clone the project

```bash
  git clone https://github.com/titoyudha/DBank/tree/master
```

Go to the project directory

```bash
  cd DBank
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  dfx start
  dfx deploy
```
